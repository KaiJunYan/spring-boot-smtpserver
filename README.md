# spring-boot-smtpserver

spring-boot   smtpserver
基于Spring Boot 和 subethasmtp实现的smtp服务器

### 主要功能
  - 实现了smtp服务器的本地启动
  - 实现了smtp协议和Spring Boot集成
  - 实现了smtp服务器的邮件内容流的保存
  - 实现了邮件格式EML文件的解析
